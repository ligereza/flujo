# Estados de job

Estados recomendados para `jobs/.../brief.yaml`:

```txt
borrador
brief_extraido_pendiente_revision
pendiente_datos
listo_para_disenar
en_diseno
generado
entregado
pausado
cancelado
```

## Uso

- `borrador`: recién creado.
- `brief_extraido_pendiente_revision`: generado automáticamente desde correo.
- `pendiente_datos`: falta info crítica.
- `listo_para_disenar`: brief aprobado, puede crearse proyecto.
- `en_diseno`: proyecto/config en preparación.
- `generado`: outputs creados.
- `entregado`: enviado o listo final.
- `pausado`: detenido temporalmente.
- `cancelado`: no continuar.
